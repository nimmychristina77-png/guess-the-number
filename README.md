# 🎯 Guess The Number

A simple game where the computer picks a number… and you try to figure it out before time runs out.

##  About

I built this as a small interactive project using plain HTML, CSS, and JavaScript.
The idea is basic — guess the number — but I added a few things to make it more fun and less predictable.

You’re not just guessing… you’re racing against time, using hints, and trying to beat your own score.

##  Features

- Difficulty levels (Easy, Medium, Hard)
- 60-second timer that adds pressure
- Hints (even/odd + range clue)
- Limited attempts based on difficulty
- Score + best score tracking
- Leaderboard saved in browser
- Sound effects (can turn on/off)
- Guess history + “warm/cold” feedback
- Clean UI with small animations and a win celebration

##  How to play

1. Open the file in your browser
2. Enter your name
3. Pick a difficulty
4. Start guessing
5. Use the hints and feedback
6. Try to get it before time or attempts run out

##  Built with
